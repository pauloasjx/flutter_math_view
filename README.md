# flutter_math_view

LaTeX inside Flutter using PlatformView

## Install
Inside pubspec.yaml add:
```yaml
flutter_math_view: ^0.0.2
```

## Usage
```dart
FlutterMathView(
	latex: "x = \\frac{-b \\pm \\sqrt{b^2-4ac}}{2a}",  
	color: Colors.purple,  
	fontSize: 32.0,  
),
```
